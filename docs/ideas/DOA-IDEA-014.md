## Title

Discovery Observability Debug View

---

## Problem

Observability данные доступны, но не имеют удобного способа просмотра.

Анализ pipeline требует доступа к данным без написания дополнительного кода.

---

## Idea

Добавить debug view для observability:

- CLI вывод стадий pipeline
- просмотр данных до и после фильтрации
- возможность сравнения стадий

---

## Scope

В рамках:

- вывод данных observability
- базовое форматирование
- CLI интерфейс

Вне scope:

- веб UI
- редактирование данных
- автоматическая аналитика
